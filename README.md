## Sorting Hat Project Overview
An app that randomly assigns you to a hogwarts wizard house after you fill out the form.

## Features
- You can get sorted into a random hogwarts house
- you can delete students cards and they are moved to the expelled students list to the right
- You can filter by the house that each student is in
